About This Demo

This interactive demo uses a Fourier Neural Operator (FNO) to predict steady-state flow around NACA airfoils. The model was trained on data generated from a finite-difference Navier-Stokes solver and can predict complex flow patterns in milliseconds, orders of magnitude faster than traditional CFD methods.

Instructions: Adjust the Reynolds number slider and select an airfoil type to see how the flow field changes. Use the view buttons to switch between different field visualizations.
